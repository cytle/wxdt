# wxdt

微信开发者工具, 作为npm包

## CONTRIBUTING

```
.
├── README.md
├── bin
│   ├── upgradeWxdt.js      // 更新package.nw源代码脚本

├── lib
│   └── updatePackageJson.js
├── package-lock.json
├── package.json
├── packages                // 为兼容修改后的包
│   └── node-sync-ipc
└── yarn.lock
```
